# 健康食谱推荐系统 python680

## 项目概述

健康食谱推荐系统是基于Python语言，结合Flask框架和Vue.js的前端技术，以及MySQL数据库构建的一套推荐系统。旨在为用户提供个性化的健康食谱推荐，提高饮食健康水平。

## 技术栈

- 后端：Python + Flask
- 前端：Vue.js
- 数据库：MySQL

## 系统角色

- 管理员：负责系统的维护和用户管理。
- 用户：使用系统进行食谱推荐，并根据自己的健康情况调整饮食。
